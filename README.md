# 🎬 Smartflix - Smart Movie Recommender System

A content based movie recommender system that suggests 
similar movies based on your favourite movie!

## 🚀 Live Demo
[Click Here to Try Smartflix](#) (add link after deployment)

## 🎯 Features
- Search from 5000+ movies
- Get 5 similar movie recommendations instantly
- Movie posters fetched from TMDB API
- Clean and simple UI

## 🛠️ Tech Stack
- Python
- Pandas & Numpy
- Scikit-learn
- NLTK
- Streamlit
- TMDB API

## 🧠 How It Works
1. Movie data is cleaned and preprocessed
2. Tags are created from genres, keywords, cast and director
3. CountVectorizer converts tags into vectors
4. Cosine Similarity finds similar movies
5. Top 5 similar movies are recommended

## 📊 Dataset
- TMDB 5000 Movies Dataset from Kaggle
- 5000+ movies with genres, keywords, cast and crew

## 👨‍💻 How To Run Locally
1. Clone the repository
2. Install requirements
pip install -r requirements.txt
3. Run the app
streamlit run app.py
